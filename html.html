¿Que es HTML?

Significa HyperText Markup Language (Lenguaje de marcado de hipertexto), es el componente más básico y en el se define el contenido de las páginas web. Básicamente se trata de un conjunto de etiquetas que sirven para definir el texto y otros elementos que compondrán una página web, como imágenes, listas, vídeos, etc.

HTML consiste en una serie de elementos, que usarás para encerrar diferentes partes del contenido con el fin de que se vean o se comporten de una manera determinada.

¿Como utilizo HTML?

¿Que son las elementos HTML?

El código HTML se compone por diferentes etiquetas que el navegador interpretará y a través de las cuales puede crear o modificar la apariencia de un documento.

Las etiquetas HTML son fragmentos de código que nos permiten crear elementos HTML. Los elementos son la estructura básica de HTML. Dichos elementos tienen dos propiedades básicas: atributos y contenido.

Un elemento, para que sea válido, generalmente tiene una etiqueta de inicio (parte azul en la imagen) y una etiqueta de cierre (parte azul en la imagen).

Como podemos ver en el ejemplo, existe una etiqueta de apertura, y una etiqueta de cierre. El texto que hay en medio de ambas etiquetas, “Esto es un ejemplo” (parte verde) se conoce como contenido del elemento. Aparte de las etiquetas y el contenido, un elemento puede tener atributos. Los atributos son la forma en que los creadores o desarrolladores web definen las propiedades de un elemento. Los atributos deben insertarse en forma de lista y separados por espacios, siempre dentro de la etiqueta para que sea válido. El atributo debe añadirse después del nombre del elemento o de la etiqueta html y precedidos por un espacio. Cada uno de estos elementos está compuesto por un nombre, el signo igual (=) y el valor o función que se encuentra a veces entre comillas (parte amarilla en la imagen). De manera resumina entonces los atributos tienen dos secciones:

    Nombre: El nombre identifica la información adicional que un usuario quiere agregar.

    Valor del atributo: Da más detalles que el anterior.
